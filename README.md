Trabalho de Atividade Extensionista de Jose Luciano Soares Filho

Aplicativo simples de controle de vendas para microeenprendedores centrado na no bairro Cidade dos Funcionários em Fortaleza/CE

Passo 1: Permitir instalação de apps de terceiros

Por padrão, o Android bloqueia a instalação de arquivos baixados fora da loja oficial.

1-   Abra as Configurações do seu celular.

2-   Procure por Aplicativos ou Apps.

3-   Toque em Acesso especial (geralmente dentro de "Avançado" ou no menu de três pontinhos).

4-   Selecione Instalar apps desconhecidos.

5-   Escolha o navegador de internet que você usa (por exemplo, o Google Chrome) e altere a chave para Permitir desta fonte. 

Passo 2: Baixar o APK no GitHub 

1-   Abra o navegador do seu celular e acesse o repositório do projeto.

2-   Na página principal do projeto, procure pelo app-debug.apk

3-   Toque no arquivo com extensão .apk para iniciar o download.

4-   O navegador avisará que o arquivo pode ser nocivo. Confirme em Fazer download mesmo assim. 

Passo 3: Instalar o APK

1-   Quando o download terminar, toque em Abrir na notificação do seu celular.

2-   Como alternativa, abra o aplicativo Meus Arquivos ou Gerenciador de Arquivos do seu celular, vá na pasta Downloads e toque no arquivo .apk.

3-   Uma tela do sistema aparecerá perguntando se deseja instalar o aplicativo. Toque em Instalar.
